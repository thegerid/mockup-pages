# Demo SFA Go

Единый demo-проект для SFA Go с одной точкой входа.

Структура:

- `index.html` — единственная внешняя точка входа в Demo SFA Go.
- `debit-card/` — текущий основной цельный сценарий проекта.
- `meetings/` — задел под будущий единый экран встреч.
- `meeting-transfer/` — задел под будущий отдельный flow переноса встречи.

Правило расширения:

- каждый новый продукт или процесс добавляется отдельной подпапкой;
- внешняя ссылка остается одной: `Demo SFA Go/index.html`;
- пользователь не должен выбирать flow на стартовом экране, пока не появится отдельная задача на такой сценарий.
